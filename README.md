# EXIT - You are the exit liquidity

A provocative minimalist memecoin landing page designed to reinforce the meme in 5 seconds.

## Concept
The web doesn't inform; it provokes. It feels like a signal, a warning, and something uncomfortable but curious.

## Features
- **Technical Dashboard UI:** High-precision terminal aesthetic with CRT scanlines.
- **Dynamic Counter:** Real-time tracking of "early" believers.
- **Provocative Interaction:** The EXIT button transitions to "TOO LATE" upon interaction.
- **System Messages:** Randomized system warnings and status updates.

## Tech Stack
- React 19
- Tailwind CSS 4
- Motion (Framer Motion)
- Lucide Icons
- Vite
